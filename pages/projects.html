RLDec

Affiliation: Université Laval (May 2023)

This project is an implementation of the RLDec decomposition approach as described in the paper "Extracting Microservices from Monolithic Systems using Deep Reinforcement Learning" .
RLDec is a decomposition tool that analyzes the source code of a monolithic Java application and suggests the recommended microservices for each class in the system using a Deep Reinforcement Learning based method. The approach was implemented mainly Using PyTorch and Ray-RLlib, a reinforcement learning library that provides a high-level API for distributed reinforcement learning.

HyDec

Affiliation: Université Laval (December 2022)

This project is an implementation of the HyDec and HierDec decomposition approaches as described in the papers:

  • HyDec: "Combining Static and Dynamic Analysis to Decompose Monolithic Application into Microservices"
  • HierDec: "A Hierarchical DBSCAN Method for Extracting Microservices from Monolithic Applications"

HierDec is a decomposition tool that analyzes the source code of a monolithic Java application through static analysis and a TFIDF based pipeline and suggests the recommended microservices for each class in the system using a hierarchical version of the DBSCAN algorithm. HyDec is an extension of the HierDec approach that combines the static analysis results with the dynamic analysis results to improve the quality of the decomposition.
The current implementation is a generalization of the approach proposed in HyDec enabling the use of various representations of the source code (with or instead of the static and dynamic analysis results). It includes the default configurations which correspond to the HyDec and HierDec papers.

Automatic classification of sunflower seeds based on X-ray images

Affiliation: Vilmorin Mikado (March 2020 - August 2020)

This project was done as part of my end of studies internship at Vilmorin Mikado. The goal was to develop a machine learning model that can classify sunflower seeds based on X-ray images. The project involved 3 main parts:

  • Developing a segmentation module to extract the seeds from the X-ray images using traditional image processing techniques
  • Building the classification module which included researching the best model to use, data preprocessing, and fine-tuning the Convolutional Neural Network (CNN) model on the DICOM image data
  • Developing a user-friendly interface to allow the users to upload the images, review them, edit them, get the classification results and fine-tune the generated results
